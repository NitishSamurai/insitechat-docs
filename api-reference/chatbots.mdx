# Chatbots

Use these endpoints to list your chatbots and retrieve details for a specific chatbot.

## List Chatbots

Returns a list of all chatbots in your account.

### Request

```
GET /v1/chatbots
```

```bash
curl https://backend.insitechat.ai/api/v1/chatbots \
  -H "Authorization: Bearer sk-your-api-key-here"
```

### Response

```json
{
  "chatbots": [
    {
      "id": "bot_abc123",
      "name": "Support Bot",
      "description": "Customer support chatbot",
      "status": "active",
      "source_count": 5,
      "message_count": 1240,
      "created_at": "2026-03-15T10:30:00Z",
      "updated_at": "2026-04-01T14:20:00Z"
    },
    {
      "id": "bot_def456",
      "name": "Sales Assistant",
      "description": "Pre-sales Q&A bot",
      "status": "active",
      "source_count": 3,
      "message_count": 870,
      "created_at": "2026-03-20T09:00:00Z",
      "updated_at": "2026-04-05T11:15:00Z"
    }
  ]
}
```

## Get Chatbot Details

Returns full details for a single chatbot.

### Request

```
GET /v1/chatbots/{id}
```

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | **Required.** The chatbot ID (e.g., `bot_abc123`). |

```bash
curl https://backend.insitechat.ai/api/v1/chatbots/bot_abc123 \
  -H "Authorization: Bearer sk-your-api-key-here"
```

### Response

```json
{
  "id": "bot_abc123",
  "name": "Support Bot",
  "description": "Customer support chatbot",
  "status": "active",
  "model": "gpt-4o-mini",
  "temperature": 0.7,
  "system_prompt": "You are a helpful customer support assistant for Acme Inc.",
  "source_count": 5,
  "message_count": 1240,
  "lead_capture_enabled": true,
  "quick_prompts": [
    "What's your return policy?",
    "How do I track my order?",
    "Talk to a human"
  ],
  "created_at": "2026-03-15T10:30:00Z",
  "updated_at": "2026-04-01T14:20:00Z"
}
```

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique chatbot identifier. |
| `name` | string | Display name of the chatbot. |
| `description` | string | Short description. |
| `status` | string | `active` or `paused`. |
| `model` | string | The AI model used for responses. |
| `temperature` | number | Response creativity (0.0 to 1.0). |
| `system_prompt` | string | The chatbot's system instructions. |
| `source_count` | integer | Number of training sources. |
| `message_count` | integer | Total messages received. |
| `lead_capture_enabled` | boolean | Whether the lead capture form is active. |
| `quick_prompts` | array | List of quick prompt strings. |
| `created_at` | string | ISO 8601 creation timestamp. |
| `updated_at` | string | ISO 8601 last-updated timestamp. |

<Info>
Chatbot IDs are prefixed with `bot_` followed by a unique alphanumeric string. You can find a chatbot's ID in the dashboard URL or in the list response above.
</Info>
