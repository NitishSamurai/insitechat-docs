# Introduction

InsiteChat enables you to build an AI chatbot that answers customer questions using your own content. The platform requires no developer involvement and setup takes approximately 5 minutes.

## What InsiteChat Does

<CardGroup cols={2}>
  <Card title="Website Crawling" icon="globe">
    Automatically indexes pages from a provided URL with customizable sync scheduling.
  </Card>
  <Card title="Document Uploads" icon="file-arrow-up">
    Supports PDF, Word, PowerPoint, plain text, Markdown, and CSV formats.
  </Card>
  <Card title="Cloud Integration" icon="cloud">
    Syncs with Google Drive, Notion, or Dropbox.
  </Card>
  <Card title="Lead Capture" icon="user-plus">
    Collects visitor contact information within the chat widget.
  </Card>
  <Card title="Conversation Analytics" icon="chart-line">
    Dashboard for reviewing customer inquiries.
  </Card>
  <Card title="Multilingual Support" icon="language">
    Speaks 95+ languages with automatic language detection.
  </Card>
  <Card title="Multi-Provider AI" icon="microchip">
    Supports Google Gemini, OpenAI ChatGPT, or self-hosted Ollama with provider switching capability.
  </Card>
</CardGroup>
